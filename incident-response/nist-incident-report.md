# NIST Incident Response Report

## Incident Summary

An employee received a phishing email that appeared to come from the company's IT department. The employee clicked a malicious link and entered login credentials.

An attacker gained unauthorized access to the employee account.

## Identify

- Phishing email received
- Employee credentials compromised
- Unauthorized access detected

## Protect

- Reset affected account passwords
- Enabled multi-factor authentication
- Updated email filtering policies

## Detect

- Login activity reviewed
- Suspicious IP address identified
- Security alerts generated

## Respond

- Disabled compromised account
- Removed attacker access
- Notified management

## Recover

- Restored secure account access
- Conducted user awareness training
- Reviewed security controls

## Lessons Learned

- MFA should be enforced for all users
- Employee phishing awareness training is critical
- Monitoring should be continuously improved
