# Incident Response Project

This project demonstrates my understanding of the NIST Cybersecurity Framework and incident response procedures.

## Skills Demonstrated
- Incident analysis
- Security documentation
- Risk assessment
- NIST Cybersecurity Framework
- Professional reporting

## Scenario
A company experienced a phishing attack that led to unauthorized access to employee accounts.

The incident was analyzed using the NIST Cybersecurity Framework:
- Identify
- Protect
- Detect
- Respond
- Recover
